## 🎓 Key Learnings Summary

### ✅ Week 1 — Excel
- [ ] SUMIF, COUNTIF, AVERAGEIF for business metrics
- [ ] VLOOKUP & XLOOKUP for data merging
- [ ] Pivot Tables for instant data summarization
- [ ] IF, AND, OR for data classification & flagging
- [ ] Data cleaning with TRIM, PROPER, IFERROR
- [ ] Building interactive dashboards with slicers

### ✅ Week 2 — SQL
- [ ] SELECT, WHERE, ORDER BY, LIMIT
- [ ] GROUP BY + HAVING for aggregated reports
- [ ] INNER JOIN, LEFT JOIN for multi-table queries
- [ ] Subqueries for complex filtering
- [ ] Window functions: RANK, ROW_NUMBER, LAG, LEAD
- [ ] CTEs for readable complex queries

### ✅ Week 3 — Python
- [ ] Pandas: read_csv, head, info, describe
- [ ] Filtering, sorting, groupby in DataFrames
- [ ] Handling nulls, duplicates, data type issues
- [ ] Matplotlib & Seaborn for visualizations
- [ ] Full Exploratory Data Analysis (EDA) workflow

### ✅ Week 4 — Power BI
- [ ] Connecting data sources & Power Query
- [ ] DAX: CALCULATE, SUMX, DIVIDE, DATESINPERIOD
- [ ] Building KPI cards, bar charts, line charts, maps
- [ ] Slicers, drillthrough, tooltips for interactivity
- [ ] Publishing & sharing dashboards
